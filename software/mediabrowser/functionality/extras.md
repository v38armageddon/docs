| Title     | Description                       |
| --------- | --------------------------------- |
| Bing Maps | Allow to watch a map of the world |

### Controls for Xbox users

| Control    | Description       |
| ---------- | ----------------- |
| A          | Enter to Bing Map |
| B          | Exit the Bing Map |
| LT         | Zoom              |
| RT         | Unzoom            |
| Left Stick | Move the map      |
