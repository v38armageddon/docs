# Pictures

{% hint style="warning" %}
You need to give the following permissions to the apps:

* Pictures
* Camera
{% endhint %}

| Title       | Description                                                                                    |
| ----------- | ---------------------------------------------------------------------------------------------- |
| My Pictures | Allow to show pictures from your local files (Windows / Xbox) and network files (Windows only) |
| Camera      | Allow to show your camera                                                                      |
