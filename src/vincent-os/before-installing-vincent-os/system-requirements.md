# System Requirements
Vincent OS requires the following minimum system specifications for installation:
## CPU
Minimum: AMD Athlon 3000G | Intel Core i5-1135G7\
Recommended: AMD Ryzen 5 5500 | Intel Core i5-12400

## RAM
Minimum: 4 GB\
Recommended: 8 GB

## Storage
Minimum: 20 GB free space\
Recommended: 50 GB free space
