## Controls for Xbox users

| Control    | Description       |
| ---------- | ----------------- |
| A          | Enter to Bing Map |
| B          | Exit the Bing Map |
| LT         | Zoom              |
| RT         | Unzoom            |
| Left Stick | Move the map      |
