# WebSM Versions
### WebSM
#### Windows
WebSM on Windows is the first version available on the Microsoft Store.

##### Functionality
- **Microsoft WebView2**: WebSM for Windows is based on Microsoft WebView2 which is based on Microsoft Edge.
- **Minimalist**: WebSM for Windows is designed to be minimalist and simple for the users, only the basic search here, no more no less.
- [**Sidebar**](functionality/sidebar.md): A Sidebar for quick search or even, listening to music.

##### System Requirements for Windows
- Operating System: Windows 10 version 19041.0 or higher
- Architecture: x64
- RAM: 1 gigabyte (GB) minimum, 2 GB recommended
- Hard Disk Space: At least 32.4 MB of available disk space

### WebSM Lite
### Windows
WebSM Lite on Windows is a very minimalist version of the WebSM project, specifically designed to optimize performance on low-end devices while retaining the same core structure.

##### Functionality
- **Microsoft WebView2**: WebSM Lite for Windows is based on Microsoft WebView2 which is based on Microsoft Edge.
- **Low computer friendly**: WebSM Lite for Windows keeps only the essential features for navigating the web, making it suitable for low-end devices.

##### System Requirements for Windows
- Operating System: Windows 10 version 19041.0 or higher
- Architecture: x64
- RAM: 1 gigabyte (GB) minimum, 2 GB recommended
- Hard Disk Space: At least 27.5 MB of available disk space
